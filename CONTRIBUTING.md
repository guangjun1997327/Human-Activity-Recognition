# Contributing to Human Activity Recognition (HAR)

First off, thank you for considering contributing to Human Activity Recognition (HAR). It's people like you that make HAR such a great tool.

## Code of Conduct
This project and everyone participating in it is governed by the [Code of Conduct](CODE_OF_CONDUCT.md). 
By participating, you are expected to uphold this code. Please report unacceptable behavior to violetxu219@gmail.com

### Pull Requests
The process described here has several goals:

- Maintain HAR's quality
- Fix problems that are important to users
- Engage the community in working toward the best possible HAR
- Enable a sustainable system for HAR's maintainers to review contributions
